# neuronal-dnam-xqtl
Code for mQTL, eQTL, colocalization, and Mendelian randomization analyses in human GLU and GABA neurons.
Colocalization and Mendelian randomization results for disease-associated CpG–gene pairs are also provided.

Repository Structure

xqtl_mapping/

Scripts for cis-mQTL and cis-eQTL mapping analyses.

coloc/

Scripts for pairwise colocalization analyses.

mr/

Scripts for one-sample Mendelian randomization analyses.

MR input files include genotype dosage matrices for clumped instrumental SNPs, residualized DNA methylation values, and residualized gene expression values. Samples are matched across input files.

results/

Colocalization and Mendelian randomization results for disease-associated CpG–gene pairs.
